## What is okfetch?

`okfetch` is a small family of TypeScript-first HTTP packages that makes `fetch` safer and more composable without hiding how the web platform works.

All request execution returns a [`Result`](https://github.com/dmmulroy/better-result) â€” success and failure are explicit data values, not thrown exceptions.

## Key features

<AccordionGroup>
  <Accordion title="Result-based error handling">
    `okfetch` never throws for expected HTTP, timeout, validation, or parsing failures. Every call resolves to a `Result<T, E>` that you can pattern-match with `.isOk()`, `.isErr()`, `.map()`, and `.match()`.
  </Accordion>
  <Accordion title="Zod-powered validation">
    Validate response bodies with `outputSchema`, structured error payloads with `apiErrorDataSchema`, and â€” when using `@okfetch/api` â€” request `body`, `params`, and `query` before the network call goes out.
  </Accordion>
  <Accordion title="Plugin lifecycle hooks">
    Extend any request flow with `init`, `onRequest`, `onResponse`, `onSuccess`, `onFail`, and `onRetry` hooks. Share cross-cutting concerns like logging, tracing, and auth across all requests.
  </Accordion>
  <Accordion title="Retries and timeouts">
    Built-in `fixed`, `linear`, and `exponential` backoff strategies. Set globally or per-request, with an optional `shouldRetry` callback for custom logic.
  </Accordion>
  <Accordion title="Auth helpers">
    First-class support for `bearer`, `basic`, and `custom` authorization header shapes â€” no manual header assembly needed.
  </Accordion>
  <Accordion title="SSE streaming">
    Set `stream: true` to receive a `ReadableStream`. If you provide an `outputSchema`, each SSE `data:` chunk is parsed and validated independently before emission.
  </Accordion>
</AccordionGroup>

## Quick example

```typescript
import { okfetch } from "@okfetch/fetch";
import { z } from "zod/v4";

const todoSchema = z.object({
  completed: z.boolean(),
  id: z.number(),
  title: z.string(),
  userId: z.number(),
});

const result = await okfetch("https://jsonplaceholder.typicode.com/todos/1", {
  outputSchema: todoSchema,
});

result.match({
  ok: (todo) => console.log(todo.title),
  err: (error) => console.error(error._tag, error.message),
});
```
